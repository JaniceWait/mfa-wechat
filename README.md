## 在线MFA验证器
### 1. 功能介绍
#### 1.1 多种激活方式
二维码扫码激活、文本激活码激活
### 1.2 多令牌绑定
支持绑定同时绑定和激活多个令牌，支持令牌便签功能，方便识别和管理令牌
### 1.3 谷歌令牌
支持谷歌令牌，可以扫描谷歌令牌二维码
